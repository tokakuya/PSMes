### Build

Tips: Mesは現在開発途中なのでプロジェクト参照は各自の環境に合わせて適宜変更してください。

以下は環境例です。

#### 事前処理
```
mkdir Mes
cd Mes

git clone https://github.com/tokakuya/Mes.git

git clone https://github.com/tokakuya/PSMes.git

```

#### ビルド(workdir=Mes/PSMes)

```

dotnet build

```
